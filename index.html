Mechanics

Designed for distinct performance

Traditional exact-match caches fail on LLMs due to slight wording adjustments. EchoCache maps inputs into vector spaces to capture semantic equivalence.

Vector Similarity Lookup

EchoCache maps prompt text through sentence embeddings, executing vector distance operations to identify matches at a high confidence score threshold (e.g. 0.92) even if the syntax is completely rephrased.

// Vector payload similarity comparison
prompt1: "Explain quantum mechanics for kids"
prompt2: "How would you explain quantum physics to a child?"
Similarity Score: 0.95 (Cache Hit)

int8 Quantized Payloads

Compresses vector dimensions using 8-bit scalar quantization. Fits 4x more items inside local cache indexes, minimizing memory usage and preventing system latency degradation.

Uncompressed 0K keys
Quantized int8 0K keys
● Uncompressed: 250K max ● int8: 1.0M max (4x)

Parallel Background Threading

EchoCache pushes new completions to Qdrant asynchronously inside background processes. Clients receive generated answers instantly; writes never block the thread runtime.

1. Client Prompt -> SDK
2. LLM miss -> Generate Response (1800ms)
3. Return answer to user (instant)
4. Background sync payload to Qdrant (parallel thread)

Zero boilerplate setup

Import, initialize, and query. EchoCache wraps existing SDK clients cleanly.

1
Install the dependency package into your codebase.
2
Initialize the Client with your validated API credential.
3
Query semantic hits using `.ask()`.
import { EchoCache } from "echocache-node";

// Initialize EchoCache client middleware
const cache = new EchoCache("ec_prod_d880430bbc14...");

// Define your standard client query execution function
async function openaiWrapper(prompt) {
  const completion = await openai.chat.completions.create({
    model: "gpt-4o",
    messages: [{ role: "user", content: prompt }]
  });
  return completion.choices[0].message.content;
}

// Resolve query using semantic caching
const response = await cache.ask(
  "What is the capital of India?",
  openaiWrapper
);
